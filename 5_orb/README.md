# 简介
这套策略的核心思想是：在美股开盘后，利用前 5 分钟形成的价格区间 (Opening Range) 作为“市场方向早期信号”，并从当日最活跃的股票中挑选最可能出现趋势走势的股票；当价格向上或向下突破这个开盘区间时入场；同时以 ATR 为基础设置止损，并在日终强制平仓，以控制风险与隔夜不确定性。
策略通过“高相对成交量筛选 + ORB 突破入场 + 固定风险止损 + 仅日内持仓”的结构，在市场最活跃的时段寻找最确定的走势，最大化利用短线趋势的爆发性，同时严格限制回撤和交易风险。
